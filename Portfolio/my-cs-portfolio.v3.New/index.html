COMPUTATIONAL BIOLOGY

DESCRIPTION
In this course I did learn about recent problems in computational biology and bioinformatics. This course did cover following topics, such as algorithms for genomic sequencing, indexing and searching in DNA string, and protein structure prediction.

Z-Algorithm is linear-time pattern searching algorithm. Z-Algorithm used to find all occurrences of a pattern in a text in a linear time basically it is used to find matches for a pattern of length in a string of length. The total time taken is O(m + n) with linear space complexity where length of text is 'n' and the length of is 'm'.

Global Sequence Alignment algorithm or Needleman–Wunsch algorithm is utilizing a dynamic programming technique to compute to find all possible alignments between two strings having the highest score by assigning a score to every possible alignment. It is used in bioinformatics to find similarities between two genomes.

Center Star algorithm is an approximation algorithm partially based on the Global Sequence Alignment algorithm. This algorithm is used to create optimal multiple alignment between multiple strings. In general it helps to predict a protein structure or it can be used to align genes/proteins from multiple species.
